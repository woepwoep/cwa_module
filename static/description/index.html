Cwa product module

requirements

For the import to work certain options must be checked.


  • Settings>Config>Sales>Allow using different units of measure
  • Settings>Config>Accounting>Features>Full Accounting
  • Settings>Config>Accounting>Sale>Analytic accounting for sales
Accounting may be called Invoicing

Setting parameters

To setup your ftp server parameters you must change the Value shown in the image.

  • ftpaddress : The address of the ftp server
  • username : Ftp login info
  • password: Ftp login info
  • tododir: The path to the directory (on the ftp server) in which the files to be parsed reside.
  • donedir: The path to the directory (on the ftp server) in which the files should be placed after being parsed.

  • File paths are dependent on the ftp server.
        For example if the server uses ~/ for home dir, paths start with ~/. (e.g. ~/Biopunt)
  • All values are separated by a semicolon!
